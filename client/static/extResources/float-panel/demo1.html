Float Panel

Scroll your screen to see the right navigation bar is always present at the top of the visible part of the screen, regardless of the scroll position.

  • Pure JavaScript(3kb). jQuery is not required.
  • Cross platform and browser support
  • Completely FREE to use.

 

How to float:left (or right)?

The sticky feature is actually the {position:fixed;} style set through the script float-panel.js.

If you want to float the panel to the left or right (like this demo that floats the panel to the right), you need to wrap the float panel with a DIV element, and style the DIV to float:left or float:right.

Do not set float:left (or float:right) style directly to the Float Panel as it will make the panel not positioned properly when it becomes position:fixed.

 

Responsive Layout

Resize the window size to smaller than 800px in width, and you will see both the sticky feature and the float:right style are gone. It is the responsive layout we want for mobile screens. This can be easily configured with the CSS @media queries:

@media only screen and (max-width: 800px) {
    .floatWrapper {float:none!important;}       
    .float-panel {position:static!important;}
    .main-content {padding:20px;margin-right:0px;}
}
